 Zomato Restaurant Success Predictor
Predicted restaurant ratings across 50k+ Zomato Bangalore listings using ML and NLP techniques.
Results

R² Score: 0.92
RMSE: 0.12
Dataset: 41,000+ restaurants

Tech Stack

Python, Pandas, NumPy
Scikit-learn, Random Forest, XGBoost
WordCloud, Streamlit

Key Steps

Cleaned and preprocessed 51k raw Zomato listings
Performed location-based EDA across Bangalore zones
Built NLP pipeline using TF-IDF on cuisine and review data
Trained Random Forest and XGBoost; selected RF with R² of 0.92
Deployed prediction app via Streamlit

Dataset

Source: Kaggle Zomato Bangalore Restaurants
Size: 51,717 restaurants, 17 columns
